# PRODIGY ML Internship - Task 01

## House Price Prediction using Linear Regression

### Objective

Build a machine learning model to predict house prices using Linear Regression.

### Dataset

California Housing Dataset (Scikit-Learn)

### Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn

### Algorithm

Linear Regression

### Result

The model was trained successfully and achieved an R² Score of approximately 0.57. A comparison of actual and predicted house prices was visualized using a scatter plot.
